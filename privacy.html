Privacy

The Golden Rule is treating others as one would want to be treated.
We treat other people's privacy the way we want ours to be treated.

It's that simple.

Personal Data

dwyl collects the bare minimum of personal data; just name and email and only so that we can contact you. All personal data is stored encrypted with the highest level of care.

Analytics

We use a privacy-friendly Analytics tool called Plausible Analytics to collect website metrics anonymously. No personal data is shared with Analytics, ever.
